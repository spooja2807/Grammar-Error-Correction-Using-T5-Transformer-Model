# Grammar Error Correction Using T5 Transformer Model

##  Features

-  Fine-tuned **T5-small** model for grammar correction  
-  Highlights incorrect and corrected words in color  
-  Displays **confidence score** for each output  
- Evaluated using **BLEU**, **ROUGE**, and **GLEU** metrics  
- Interactive **Gradio interface** for real-time testing  
- Visualized data distributions and training performance  

---

##  Technologies Used

- **Python** – Core language  
- **Transformers (Hugging Face)** – Model loading and fine-tuning  
- **PyTorch** – Deep learning framework  
- **Pandas**, **NumPy**, **Scikit-learn** – Data preprocessing and handling  
- **Matplotlib**, **Seaborn** – Data visualization  
- **Gradio** – User interface for real-time testing  
- **Google Colab** – Training and experimentation environment
